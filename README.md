# Hospital Management System

A Java-based application that simulates core operations of a hospital environment using Object-Oriented Programming principles. The system models real-world hospital entities and their interactions, providing a structured way to manage hospital data and workflows.

---

## About the Project

This project was developed as part of an Object-Oriented Programming course. It demonstrates the practical application of OOP concepts that include inheritance, abstraction, interfaces, and polymorphism within a real-world domain: hospital management.

---

## Features

- **Patient Management**: Store and display patient information including medical history, diagnosis, treatment, and appointment records.
- **Doctor Management**: Manage doctor profiles, track availability, and handle appointment scheduling with conflict detection.
- **Staff Management**: Maintain records for nurses and administrative staff.
- **Appointment System**: Book appointments between patients and doctors with automatic time-slot validation.
- **Surgery Scheduling**: Schedule and track surgeries within hospital departments.
- **Department Management**: Organize doctors, nurses, and surgeries under named departments.
